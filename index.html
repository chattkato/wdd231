About This Course

WDD231 — Web Frontend Development builds on foundational web skills to develop dynamic, accessible, and responsive web experiences. The course covers semantic HTML, modern CSS layout techniques, and JavaScript for DOM manipulation, API consumption, and progressive enhancement.

Students apply these skills through a semester-long Chamber of Commerce project and a final capstone, both of which are linked in the navigation above.

Learning Outcomes

  • Design and develop responsive, standards-compliant websites using HTML5, CSS3, and JavaScript.
  • Apply web accessibility and SEO best practices to ensure inclusive, discoverable web content.
  • Use JavaScript to manipulate the DOM, consume APIs, and create interactive user interfaces.
  • Employ modern development workflows including Git version control and GitHub Pages deployment.
